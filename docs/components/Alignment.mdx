# Alignment

## Arguments

#### Prop Signature

```ts
type AlignmentProps = {
  children: React.ReactNode;
  vertical?: boolean;
  horizontal?: boolean;
};
```

#### children: React.ReactNode
`Alignment` is expecting to wrap `children`.

#### vertical?: boolean
`Alignment` may receive an optional `vertical` prop. If this value is passed in, then the outermost child element that `Alignment` wraps will be vertically aligned within `Alignment`.

#### horizontal?: boolean
`Alignment` may receive an optional `horizontal` prop. If this value is passed in, then the outermost child element that `Alignment` wraps will be horizontally aligned within `Alignment`.

## State
`Alignment` does not manage any state.

## Example Usage

This example is taken from the **Login** page. Here, `Alignment` wraps a Fragment child and is passed the optional `vertical` and `horizontal` props so that the child will be centered vertically and horizontally within the `Alignment` component.

```tsx
<Alignment vertical horizontal>
  { isLoading ? <Spinner /> : (
    <>
      { didAuthFail &&
        <p style={{ color: 'red', fontSize: '1.14rem', textAlign: 'center' }}>
          Incorrect username or password.
        </p>
      }
      <Form onSubmit={ login } >
        <Input onChange={ handleInputChange('email') } placeholder='Email Address' />
        <Input onChange={ handleInputChange('password') } placeholder='Password' password />
      </Form>
    </>
  )}
</Alignment>
```
